# 🧭 NAAC Compass

NAAC Compass is a user-friendly web application designed to simplify the NAAC Binary Accreditation process for higher education institutions.

It provides a structured, visual, and interactive approach to understanding NAAC’s 10-attribute framework, preparing AQAR reports, tracking compliance, and managing institutional tasks.

---

## 🚀 Features

- 📊 10-Attribute NAAC Framework (simplified explanations)
- 📝 AQAR Writing Support with structured fields
- ✅ DVV Checklist Tracking
- 📅 NAAC Timeline & Calendar View
- 📌 Task & Progress Tracker
- 🧠 Built-in AI Assistant (for guidance)
- 🏫 Multi-college profile support
- 🌐 Clean, responsive UI

---

## 💻 Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- LocalStorage (for data persistence)

---

## ⚙️ How to Use

1. Open the app in your browser  
2. Select your role  
3. Explore NAAC attributes  
4. Use AQAR, DVV, and Tracker modules  
5. Save progress automatically (browser-based)
